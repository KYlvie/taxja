### 1. 配置Stripe测试模式

#### 步骤1: 注册Stripe账户
1. 访问 https://stripe.com
2. 注册账户（选择测试模式）
3. 完成账户验证

#### 步骤2: 创建产品和价格
在Stripe Dashboard中创建：

**Plus Plan**:
- 产品名称: "Taxja Plus"
- 月度价格: €4.90
- 年度价格: €49.00
- 记录Price ID（例如: price_xxx）

**Pro Plan**:
- 产品名称: "Taxja Pro"
- 月度价格: €9.90
- 年度价格: €99.00
- 记录Price ID（例如: price_yyy）

#### 步骤3: 配置Webhook
1. Stripe Dashboard → Developers → Webhooks
2. 添加端点: `http://localhost:8000/api/v1/webhooks/stripe`
3. 选择事件:
   - `checkout.session.completed`
   - `invoice.payment_succeeded`
   - `invoice.payment_failed`
   - `customer.subscription.updated`
   - `customer.subscription.deleted`
4. 复制Webhook签名密钥

#### 步骤4: 更新环境变量
编辑 `backend/.env`:
```env
# Stripe测试密钥
STRIPE_SECRET_KEY=sk_test_...
STRIPE_PUBLISHABLE_KEY=pk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...

# Stripe Price IDs
STRIPE_PLUS_MONTHLY_PRICE_ID=price_xxx
STRIPE_PLUS_YEARLY_PRICE_ID=price_xxx
STRIPE_PRO_MONTHLY_PRICE_ID=price_yyy
STRIPE_PRO_YEARLY_PRICE_ID=price_yyy
```
