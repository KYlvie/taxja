# Taxja 简化用户体验改进计划

## 目标
让用户能够以最简单、最直观的方式使用所有功能，而不是简化功能本身。

## 改进策略

### 1. 渐进式引导（Progressive Disclosure）
- 首次使用显示简单向导
- 高级功能默认隐藏，需要时才显示
- 每个步骤都有清晰的说明

### 2. 智能默认值
- 自动填充常用数据
- OCR 识别后自动建议操作
- 减少必填字段

### 3. 友好的错误处理
- 中文错误提示
- 提供解决方案
- 不显示技术细节

### 4. 快捷操作
- 一键完成常见任务
- 智能建议下一步操作
- 减少点击次数

## 用户旅程示例

### 新用户首次使用
1. 注册/登录
2. 看到欢迎向导："让我们开始吧！"
3. 引导上传购房合同
4. 系统自动识别并创建房产
5. 提示设置月租金（可选）
6. 完成！显示仪表板

### 日常使用
1. 上传收据/发票
2. 系统自动分类
3. 用户确认或修改
4. 自动计算税务影响

### 年度报税
1. 点击"生成年度报告"
2. 系统自动汇总所有数据
3. 显示可抵扣金额
4. 导出报告用于 FinanzOnline

## 设计原则

1. **简单优先**：默认显示最常用的功能
2. **渐进增强**：高级功能按需显示
3. **即时反馈**：每个操作都有明确的结果
4. **容错设计**：允许用户犯错并轻松纠正
5. **智能辅助**：系统主动提供帮助和建议
