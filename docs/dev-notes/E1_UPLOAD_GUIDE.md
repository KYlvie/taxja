# E1表单上传和导入指南

## 问题说明

你上传了2022年的E1表单，但没有看到任何内容显示。这是因为E1表单上传是一个**两步流程**：

1. **上传和解析**（已完成）- 提取文本和数据
2. **确认导入**（需要手动操作）- 将数据保存到数据库并创建交易

## 完整流程

### 第1步：上传E1表单

1. 进入"文档"页面
2. 点击"上传E1表单"按钮
3. 选择PDF文件
4. 系统自动提取文本并解析

**此时数据还没有保存！**

### 第2步：查看解析预览

上传成功后，你应该看到一个预览界面，显示：

- ✅ 税务年度（Tax Year）
- ✅ 纳税人姓名（Taxpayer Name）
- ✅ 税号（Steuernummer）
- ✅ 提取的所有KZ值（Kennzahlen）
- ✅ 置信度分数

**示例预览界面：**
```
📋 E1表单导入

解析预览
置信度: 85%

税务年度: 2022
纳税人姓名: [你的名字]
税号: [你的税号]

提取的KZ值:
KZ 245: €50,000.00
KZ 330: €12,000.00
...

[✅ 确认导入]  [取消]
```

### 第3步：确认导入

**重要**：你必须点击"✅ 确认导入"按钮，系统才会：
- 创建交易记录
- 保存到数据库
- 在交易列表中显示

### 第4步：查看导入结果

导入成功后会显示：
- 创建的交易数量
- 每笔交易的详细信息（类型、金额、描述、KZ编号）

然后你可以在以下位置查看数据：
- **交易页面** - 查看所有创建的交易
- **仪表板** - 查看收入/支出统计
- **税务计算** - 使用导入的数据进行计算
