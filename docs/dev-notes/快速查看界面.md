# 快速查看 Taxja 界面

## 🚀 最快方式：只启动前端

Docker 构建需要 5-10 分钟（首次），如果你想立即看到界面效果，可以先启动前端：

### 步骤 1: 打开新的终端窗口

**重要**: 不要关闭当前的 Docker 构建窗口，让它继续运行。

### 步骤 2: 启动前端开发服务器

```bash
cd frontend
npm run dev
```

### 步骤 3: 访问界面

打开浏览器访问: **http://localhost:5173**

你会看到 Taxja 的登录界面！

### 📸 你会看到什么

- ✅ 精美的登录页面
- ✅ 语言切换器（德语/英语/中文）
- ✅ 响应式设计（可以调整浏览器窗口大小测试）
- ⚠️ 无法登录（因为后端还没启动）

### 🎨 界面预览

登录页面包含：
- Taxja Logo 和 Slogan
- 邮箱和密码输入框
- 双因素认证输入框
- "记住我" 选项
- 注册链接
- 语言切换器

### ⏳ 等待 Docker 完成

当 Docker 构建完成后（大约还需要 3-5 分钟），你就可以：
1. 登录系统
2. 查看完整功能
3. 测试所有特性
