## 当前用户体验流程

### 新用户首次使用
1. **注册/登录** → 进入仪表板
2. **看到欢迎向导** → "欢迎使用 Taxja！"
3. **点击"开始使用"** → 进入第二步
4. **引导上传文档** → "上传您的第一个文档"
5. **点击"去上传文档"** → 跳转到文档页面
6. **上传购房合同** → OCR自动识别
7. **确认导入建议** → 自动创建房产记录
8. **可选设置租金** → 创建定期交易
9. **完成！** → 返回仪表板查看数据

### 日常使用
1. **上传收据/发票** → 自动OCR识别
2. **确认或修改** → 系统自动分类
3. **查看定期交易** → 管理租金和贷款
4. **查看房产** → 监控折旧和收入
5. **生成报告** → 导出用于报税

### 定期交易管理
1. **访问定期交易页面** → 看到精美的卡片列表
2. **查看现有交易** → 2条租金收入（€320和€640）
3. **筛选交易** → 全部/活跃/已暂停
4. **管理交易** → 暂停/恢复/编辑/删除
5. **创建新交易** → 租金收入或贷款利息
