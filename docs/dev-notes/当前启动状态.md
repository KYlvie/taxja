## 🔍 如何查看进度

### 查看实时日志

在另一个终端窗口运行：
```bash
docker-compose logs -f
```

### 查看容器状态

```bash
docker ps
```

当你看到类似这样的输出时，说明启动成功：
```
CONTAINER ID   IMAGE              STATUS         PORTS
xxxxx          taxja-backend      Up 2 minutes   0.0.0.0:8000->8000/tcp
xxxxx          taxja-frontend     Up 2 minutes   0.0.0.0:5173->5173/tcp
```

## ⚠️ 常见问题

### 构建时间太长？

第一次构建需要：
- 下载基础镜像（~500MB）
- 安装 Python 依赖（~200MB）
- 安装 Node 依赖（~300MB）

这是正常的，只有第一次需要这么久。

### 如何停止？

```bash
docker-compose down
```

### 如何重新启动？

```bash
docker-compose restart
```
