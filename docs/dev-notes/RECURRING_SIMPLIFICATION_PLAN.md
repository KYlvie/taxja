## 用户体验流程对比

### 优化前（复杂）：
```
1. 用户添加房产
2. 用户去"定期交易"页面
3. 点击"创建租金收入"
4. 选择房产
5. 填写表单
6. 提交
```

### 优化后（简单）：
```
1. 用户添加房产
2. 系统弹出："要自动记录租金吗？"
3. 用户输入金额，点击"启用"
4. 完成！
```

## 技术实现要点

### 1. 检测是否已设置自动收租
```typescript
// 在房产详情页加载时
const checkAutoRent = async (propertyId: string) => {
  const recurring = await recurringService.getByProperty(propertyId);
  const rentalIncome = recurring.find(r => r.recurring_type === 'rental_income');
  return rentalIncome;
};
```

### 2. 标记自动生成的交易
```typescript
// 在 Transaction 模型中添加字段
interface Transaction {
  // ... 其他字段
  is_system_generated?: boolean;
  recurring_transaction_id?: number;
}

// 在交易列表中显示
{transaction.is_system_generated && (
  <span title="由系统自动生成">🤖</span>
)}
```

### 3. 房产卖出时自动停止
```python
# backend/app/services/property_service.py
async def sell_property(property_id: UUID, sale_date: date):
    # 更新房产状态
    property.status = PropertyStatus.SOLD
    property.sale_date = sale_date
    
    # 自动停止所有相关定期交易
    await recurring_service.auto_pause_for_sold_property(property_id, sale_date)
    
    return property
```
