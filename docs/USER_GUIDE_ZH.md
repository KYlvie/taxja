## 入门指南

### 注册

1. 访问 https://taxja.at
2. 点击"注册"
3. 输入您的电子邮件和安全密码
4. 选择您的用户类型：
   - **职员**：有工资单的受薪员工
   - **房东**：有租金收入的个人
   - **个体户**：自由职业者、个体经营者
   - **混合**：多种收入来源

### 设置个人资料

注册后，完善您的个人资料：

1. **个人信息**
   - 姓名、地址
   - 税号 (Steuernummer)
   - 增值税号 (UID)（如果是个体户）

2. **家庭信息**
   - 子女数量（用于子女税收抵免）
   - 单亲状态（用于额外扣除）

3. **通勤信息**
   - 到工作地点的距离（用于通勤补贴）
   - 是否有公共交通

### 双因素认证 (2FA)

为了额外的安全性，启用 2FA：

1. 转到设置 → 安全
2. 点击"启用 2FA"
3. 使用认证器应用（如 Google Authenticator）扫描二维码
4. 输入 6 位数代码

## 管理交易

### 手动创建交易

1. 点击"交易" → "新建交易"
2. 选择类型：
   - **收入**：工资、租金收入、费用
   - **支出**：业务费用、可抵扣成本
3. 输入：
   - 金额 (€)
   - 日期
   - 描述
   - 类别（自动建议）
4. 点击"保存"

### 导入交易

#### 从银行账户导入 CSV

1. 将您的银行对账单导出为 CSV
2. 点击"交易" → "导入"
3. 从列表中选择您的银行
4. 上传 CSV 文件
5. 查看预览
6. 确认导入

**支持的银行：**
- Raiffeisen
- Erste Bank
- Sparkasse
- Bank Austria
- BAWAG P.S.K.

### 自动分类

Taxja 自动对交易进行分类：

- **基于规则**：已知商家（BILLA、SPAR 等）
- **AI 驱动**：从您的更正中学习

**更正类别：**
1. 点击交易
2. 选择正确的类别
3. Taxja 从您的更正中学习

### 检查可抵扣性

Taxja 自动显示费用是否可抵税：

- ✅ **可抵扣**：在税务计算中考虑
- ❌ **不可抵扣**：个人费用

**可抵扣费用示例：**

**个体户：**
- 办公用品
- 工作设备
- 专业书籍
- 培训课程
- 商务旅行

**房东：**
- 维护
- 物业管理
- 保险
- 房产税
- 贷款利息

## 文档上传与 OCR

### 上传文档

1. 点击"文档" → "上传"
2. 选择文件：
   - 用智能手机拍照
   - 从计算机选择文件
   - 拖放
3. 支持的格式：JPG、PNG、PDF
4. 最大大小：每个文件 10 MB

### OCR 识别

Taxja 自动识别：

**工资单 (Lohnzettel)：**
- 税前工资
- 税后工资
- 预扣税
- 社会保险

**收据：**
- 日期
- 金额
- 商家
- 明细项目
- 增值税金额

**发票：**
- 发票号码
- 日期
- 金额
- 供应商
- 增值税金额

### 审核 OCR 结果

1. 上传后，您将看到识别的数据
2. 置信度低的字段会被突出显示
3. 如需要，更正值
4. 点击"确认"
5. Taxja 自动创建交易

**更好识别的提示：**
- 良好的照明
- 平整的文档（不要折叠）
- 整个文档在画面中
- 足够的分辨率

### 个体户收据分析

对于超市收据，Taxja 会询问：

> "这是商业采购吗？"

Taxja 分析明细项目：
- ✅ 办公用品、清洁产品 → 可抵扣
- ❌ 食品、个人物品 → 不可抵扣

您可以调整选择。

## 税务计算

### 税务概览

仪表板显示：

- **年收入**：所有收入的总和
- **可抵扣费用**：所有业务费用的总和
- **预估税款**：预期税务负担
- **已支付**：预扣税、预付款
- **剩余**：应付余额

### 所得税

Taxja 使用 **官方 USP 2026 税率** 计算：

| 收入 | 税率 |
|------|------|
| €0 – €13,539 | 0% |
| €13,539 – €21,992 | 20% |
| €21,992 – €36,458 | 30% |
| €36,458 – €70,365 | 40% |
| €70,365 – €104,859 | 48% |
| €104,859 – €1,000,000 | 50% |
| 超过 €1,000,000 | 55% |

### 增值税 (VAT)

**小企业豁免：**
- 营业额 ≤ €55,000：免增值税
- 营业额 €55,000 – €60,500：容忍规则（仍然免税，但明年自动纳税）
- 营业额 > €60,500：需缴纳增值税

**增值税税率：**
- 标准：20%
- 住宅租赁：10%（或可选择豁免）

### 社会保险 (SVS)

对于个体户，Taxja 自动计算：

- 养老保险：18.5%
- 医疗保险：6.8%
- 事故保险：€12.95/月
- 补充养老金：1.53%

**最低缴费基数：** €551.10/月
**最高缴费基数：** €8,085/月

SVS 缴费可作为特殊支出抵扣！

### 税收抵免

**通勤补贴 (Pendlerpauschale)：**
- 20-40 公里：€58-€306/月
- 加上通勤欧元：€6/公里/年

**家庭办公补贴：**
- €300/年（自动）

**子女税收抵免：**
- €58.40/月/每个孩子

**单亲抵免：**
- €494/年

### 模拟器

测试税务场景：

1. 转到"税务" → "模拟器"
2. 添加假设费用
3. 立即查看税收节省
4. 示例："如果我买一台新笔记本电脑，能节省多少税？"

### 平税制 vs. 实际记账

对于小企业，Taxja 显示两种选项：

**平税制 (Pauschalierung)：**
- 6% 或 12% 固定费用扣除
- 更简单，工作量更少
- 无需收据

**实际记账：**
- 实际费用可抵扣
- 工作量更大，需要收据
- 通常税收节省更多

Taxja 推荐更便宜的选项！

## 生成报表

### 创建税务申报表

1. 转到"报表" → "创建新报表"
2. 选择税务年度
3. 选择格式：
   - **PDF**：综合报告
   - **XML**：用于 FinanzOnline
   - **CSV**：用于 Excel/税务顾问
4. 选择语言（德语、英语、中文）
5. 点击"生成"

### PDF 报告

PDF 报告包含：

- 个人信息
- 收入概览
- 支出概览
- 税务计算（详细）
- 税收抵免
- 摘要

### FinanzOnline XML

1. 生成 XML 报告
2. 下载文件
3. 登录 FinanzOnline
4. 上传 XML 文件

**重要：** Taxja 根据官方 FinanzOnline 架构验证 XML！

### 审计清单

提交前检查：

1. 转到"报表" → "审计清单"
2. Taxja 显示：
   - ✅ 所有交易已记录
   - ⚠️ 缺少 2 张收据
   - ✅ 所有扣除已记录
3. 提交前解决警告

## FAQ

### 一般问题

**问：Taxja 能替代税务顾问吗？**
答：不能。Taxja 是一个税务管理工具。对于复杂情况，我们建议咨询税务顾问。

**问：我的数据安全吗？**
答：是的。Taxja 使用 AES-256 加密存储数据，使用 TLS 1.3 传输。我们符合 GDPR 标准。

**问：我可以在智能手机上使用 Taxja 吗？**
答：可以！Taxja 作为渐进式 Web 应用 (PWA) 提供，可在所有设备上使用。

### 交易

**问：如何从银行导入交易？**
答：将您的银行对账单导出为 CSV，然后在"交易" → "导入"下导入。

**问：如果我错误分类了交易会怎样？**
答：没问题！只需更正类别。Taxja 会从您的更正中学习。

**问：是否自动检测重复项？**
答：是的。导入时，Taxja 会检查重复项（相同金额、日期、相似描述）。

### OCR

**问：Taxja 可以识别哪些文档？**
答：工资单、收据、发票、租赁合同、SVS 通知、银行对账单。

**问：如果 OCR 识别失败怎么办？**
答：您可以手动输入数据。更好识别的提示：良好的照明、平整的文档、足够的分辨率。

**问：原始文档是否保存？**
答：是的。所有上传的文档都经过加密存储，随时可访问。

### 税务

**问：税务计算有多准确？**
答：Taxja 使用官方 USP 2026 税率。计算已根据官方 USP 计算器验证（偏差 < €0.01）。

**问：我可以计算往年的税款吗？**
答：可以。Taxja 支持多个税务年度和亏损结转。

**问：工资税和所得税有什么区别？**
答：工资税由雇主预扣。所得税是所有收入的总税款。通过员工税务评估 (Arbeitnehmerveranlagung)，您可以要求退还多付的工资税。

### 报表

**问：我可以直接将报表发送到 FinanzOnline 吗？**
答：不可以。您必须下载 XML 文件并手动上传到 FinanzOnline。（FinanzOnline 没有公共 API。）

**问：我可以用哪些语言创建报表？**
答：德语、英语和中文。

**问：我可以将报表发送给我的税务顾问吗？**
答：可以。导出为 PDF 或 CSV，然后通过电子邮件发送文件。

### 隐私

**问：我可以导出我的数据吗？**
答：可以。在"设置" → "导出数据"下，您可以将所有数据下载为 ZIP 存档（GDPR 权利）。

**问：我可以删除我的账户吗？**
答：可以。在"设置" → "删除账户"下，您的所有数据将被永久删除。

**问：谁可以访问我的数据？**
答：只有您。Taxja 员工无法访问您的税务数据。
